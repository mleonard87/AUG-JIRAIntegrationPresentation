APIs

  • Application Programming Interface - general intro to what an API is
  • What are they? (JIRA specific)
  • SOAP vs REST (which JIRA version did each get introduced)
  • Fantastic documentation
  • Libraries available (official java, 3rd party python, etc)
  • Available self hosted and on demand but need to be enabled.
  • Why are they good? - There pretty much an API available for every oepration you can do within JIRA so everything can be automated or intergrated with whatever you want
